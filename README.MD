## **–ü—ñ–¥ –∫–∞–ø–æ—Ç–æ–º: –°–ø—Ä–∞–≤–∂–Ω—è –º–∞–≥—ñ—è ‚Äî `size` –ø—Ä–æ—Ç–∏ `capacity`**

–û—Å—å –º–∏ –π –¥—ñ—Å—Ç–∞–ª–∏—Å—è –¥–æ –Ω–∞–π—Ü—ñ–∫–∞–≤—ñ—à–æ–≥–æ. –Ø–∫ "—á–∞—Ä—ñ–≤–Ω–∏–π —Ä—é–∫–∑–∞–∫" —Ä–æ–∑—à–∏—Ä—é—î—Ç—å—Å—è? –£—è–≤—ñ—Ç—å, —â–æ –≤–∞—à –≤–µ–∫—Ç–æ—Ä ‚Äî —Ü–µ **—Ä–µ—Å—Ç–æ—Ä–∞–Ω**.

  * **`size()`** ‚Äî —Ü–µ **–∫—ñ–ª—å–∫—ñ—Å—Ç—å –∑–∞–π–Ω—è—Ç–∏—Ö —Å—Ç–æ–ª–∏–∫—ñ–≤**. –¶–µ —Å–∫—ñ–ª—å–∫–∏ –∫–ª—ñ—î–Ω—Ç—ñ–≤ –∑–∞—Ä–∞–∑ —Å–∏–¥–∏—Ç—å —É –∑–∞–ª—ñ.
  * **`capacity()`** ‚Äî —Ü–µ **–∑–∞–≥–∞–ª—å–Ω–∞ –∫—ñ–ª—å–∫—ñ—Å—Ç—å —Å—Ç–æ–ª–∏–∫—ñ–≤ —É —Ä–µ—Å—Ç–æ—Ä–∞–Ω—ñ**. –¶–µ —Å–∫—ñ–ª—å–∫–∏ –∫–ª—ñ—î–Ω—Ç—ñ–≤ —Ä–µ—Å—Ç–æ—Ä–∞–Ω *–º–æ–∂–µ* –≤–º—ñ—Å—Ç–∏—Ç–∏, –ø–µ—Ä—à –Ω—ñ–∂ –¥–æ–≤–µ–¥–µ—Ç—å—Å—è –¥–æ–±—É–¥–æ–≤—É–≤–∞—Ç–∏ –Ω–æ–≤—É –∑–∞–ª—É.

–ö–æ–ª–∏ –≤–∏ —Å—Ç–≤–æ—Ä—é—î—Ç–µ –≤–µ–∫—Ç–æ—Ä, –≤—ñ–Ω –º–æ–∂–µ –±—É—Ç–∏ –ø–æ—Ä–æ–∂–Ω—ñ–º, –∞–ª–µ –∫–æ–º–ø'—é—Ç–µ—Ä –≤–∂–µ –≤–∏–¥—ñ–ª–∏–≤ –ø—ñ–¥ –Ω—å–æ–≥–æ —Ç—Ä–æ—Ö–∏ –º—ñ—Å—Ü—è "–Ω–∞ –≤–∏—Ä—ñ—Å—Ç".

**–©–æ –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è, –∫–æ–ª–∏ –∑–∞–∫—ñ–Ω—á—É—é—Ç—å—Å—è —Å—Ç–æ–ª–∏–∫–∏?**
–ö–æ–ª–∏ –ø—Ä–∏—Ö–æ–¥–∏—Ç—å –Ω–æ–≤–∏–π –∫–ª—ñ—î–Ω—Ç (`push_back`), –∞ –≤—Å—ñ —Å—Ç–æ–ª–∏–∫–∏ –≤–∂–µ –∑–∞–π–Ω—è—Ç—ñ (`size == capacity`), –≤—ñ–¥–±—É–≤–∞—î—Ç—å—Å—è **—Ä–µ–∞–ª–æ–∫–∞—Ü—ñ—è**:

1.  –†–µ—Å—Ç–æ—Ä–∞–Ω –∑–Ω–∞—Ö–æ–¥–∏—Ç—å **–Ω–æ–≤–µ, –∑–Ω–∞—á–Ω–æ –±—ñ–ª—å—à–µ –ø—Ä–∏–º—ñ—â–µ–Ω–Ω—è** (–≤–∏–¥—ñ–ª—è—î—Ç—å—Å—è –Ω–æ–≤–∏–π, –±—ñ–ª—å—à–∏–π —à–º–∞—Ç–æ–∫ –ø–∞–º'—è—Ç—ñ).
2.  **–í—Å—ñ—Ö —Å—Ç–∞—Ä–∏—Ö –∫–ª—ñ—î–Ω—Ç—ñ–≤ –ø–µ—Ä–µ—Å–∞–¥–∂—É—é—Ç—å** —É –Ω–æ–≤–µ –ø—Ä–∏–º—ñ—â–µ–Ω–Ω—è (–≤—Å—ñ –µ–ª–µ–º–µ–Ω—Ç–∏ –∫–æ–ø—ñ—é—é—Ç—å—Å—è).
3.  **–°—Ç–∞—Ä–µ –ø—Ä–∏–º—ñ—â–µ–Ω–Ω—è –∑–Ω–æ—Å—è—Ç—å** (—Å—Ç–∞—Ä–∞ –ø–∞–º'—è—Ç—å –∑–≤—ñ–ª—å–Ω—è—î—Ç—å—Å—è).
4.  **–ù–æ–≤–æ–≥–æ –∫–ª—ñ—î–Ω—Ç–∞ —Å–∞–¥—è—Ç—å** –∑–∞ –≤—ñ–ª—å–Ω–∏–π —Å—Ç–æ–ª–∏–∫ —É –Ω–æ–≤–æ–º—É –ø—Ä–∏–º—ñ—â–µ–Ω–Ω—ñ.

–¶–µ –¥–æ—Ä–æ–≥–∞ –æ–ø–µ—Ä–∞—Ü—ñ—è\! –©–æ–± –Ω–µ —Ä–æ–±–∏—Ç–∏ —Ü–µ —â–æ—Ä–∞–∑—É, `vector` –∑–∞–∑–≤–∏—á–∞–π –ø–æ–¥–≤–æ—é—î —Å–≤–æ—é –º—ñ—Å—Ç–∫—ñ—Å—Ç—å (`capacity`).

## **–ü—Ä–æ—Å—É–Ω—É—Ç—ñ —ñ–Ω—Å—Ç—Ä—É–º–µ–Ω—Ç–∏: –ö–µ—Ä—É—î–º–æ —Ä–æ–∑–º—ñ—Ä–æ–º —Ç–∞ –ø–∞–º'—è—Ç—Ç—é**

–¢–µ–ø–µ—Ä, –∫–æ–ª–∏ –≤–∏ –∑–Ω–∞—î—Ç–µ –ø—Ä–æ `size` —Ç–∞ `capacity`, —Ü—ñ –º–µ—Ç–æ–¥–∏ —Å—Ç–∞–Ω—É—Ç—å –¥–ª—è –≤–∞—Å –∑—Ä–æ–∑—É–º—ñ–ª–∏–º–∏.

### **`reserve()` ‚Äî –ó–∞–±—Ä–æ–Ω—é–≤–∞—Ç–∏ —Å—Ç–æ–ª–∏–∫–∏ –∑–∞–∑–¥–∞–ª–µ–≥—ñ–¥—å**

–Ø–∫—â–æ –≤–∏ –∑–Ω–∞—î—Ç–µ, —â–æ –¥–æ –≤–∞—Å –ø—Ä–∏–π–¥–µ –≤–µ–ª–∏–∫–∞ –∫–æ–º–ø–∞–Ω—ñ—è (–Ω–∞–ø—Ä–∏–∫–ª–∞–¥, –≤–∏ –±—É–¥–µ—Ç–µ –∑—á–∏—Ç—É–≤–∞—Ç–∏ 1000 —Å—Ç—É–¥–µ–Ω—Ç—ñ–≤ –∑ —Ñ–∞–π–ª—É), –≤–∏ –º–æ–∂–µ—Ç–µ –ø–æ–ø—Ä–æ—Å–∏—Ç–∏ —Ä–µ—Å—Ç–æ—Ä–∞–Ω –ø—ñ–¥–≥–æ—Ç—É–≤–∞—Ç–∏—Å—è.

// –¢–µ–ø–µ—Ä –Ω–∞—Å—Ç—É–ø–Ω—ñ 100 –≤–∏–∫–ª–∏–∫—ñ–≤ .push_back() –±—É–¥—É—Ç—å –¥—É–∂–µ —à–≤–∏–¥–∫–∏–º–∏,
// –±–æ –Ω–µ –±—É–¥–µ –∂–æ–¥–Ω–æ—ó –¥–æ—Ä–æ–≥–æ—ó —Ä–µ–∞–ª–æ–∫–∞—Ü—ñ—ó!

–í–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–π—Ç–µ `reserve`, –∫–æ–ª–∏ –≤–∏ –∑–Ω–∞—î—Ç–µ, —Å–∫—ñ–ª—å–∫–∏ –µ–ª–µ–º–µ–Ω—Ç—ñ–≤ –≤–∞–º **–ø—Ä–∏–±–ª–∏–∑–Ω–æ** –∑–Ω–∞–¥–æ–±–∏—Ç—å—Å—è, —â–æ–± –æ–ø—Ç–∏–º—ñ–∑—É–≤–∞—Ç–∏ –ø—Ä–æ–≥—Ä–∞–º—É.

### **`resize()` ‚Äî –ó–º—ñ–Ω–∏—Ç–∏ –∫—ñ–ª—å–∫—ñ—Å—Ç—å –∫–ª—ñ—î–Ω—Ç—ñ–≤**

–¶–µ–π –º–µ—Ç–æ–¥ –∑–æ–≤—Å—ñ–º —ñ–Ω—à–∏–π. –í—ñ–Ω –∑–º—ñ–Ω—é—î —Å–∞–º–µ `size` (–∫—ñ–ª—å–∫—ñ—Å—Ç—å –∫–ª—ñ—î–Ω—Ç—ñ–≤).

    –†–µ—Å—Ç–æ—Ä–∞–Ω –¥–æ–¥–∞—î –Ω–æ–≤—ñ —Å—Ç–æ–ª–∏–∫–∏ —ñ —Å–∞–¥–∏—Ç—å –∑–∞ –Ω–∏—Ö "–∫–ª—ñ—î–Ω—Ç—ñ–≤ –∑–∞ –∑–∞–º–æ–≤—á—É–≤–∞–Ω–Ω—è–º" (–¥–ª—è `int` —Ü–µ 0, –¥–ª—è `string` ‚Äî –ø–æ—Ä–æ–∂–Ω—ñ–π —Ä—è–¥–æ–∫).

    "–ó–∞–π–≤—ñ" –∫–ª—ñ—î–Ω—Ç–∏ –ø—Ä–æ—Å—Ç–æ –π–¥—É—Ç—å –¥–æ–¥–æ–º—É (–µ–ª–µ–º–µ–Ω—Ç–∏ –≤ –∫—ñ–Ω—Ü—ñ –≤–∏–¥–∞–ª—è—é—Ç—å—Å—è).

    ```cpp

    // –¢–µ–ø–µ—Ä –≤–µ–∫—Ç–æ—Ä: {95}. size = 1.
    ```

**–ì–æ–ª–æ–≤–Ω–∞ –≤—ñ–¥–º—ñ–Ω–Ω—ñ—Å—Ç—å:** `reserve` –≥–æ—Ç—É—î –ø–∞–º'—è—Ç—å, `resize` –∑–º—ñ–Ω—é—î –∫—ñ–ª—å–∫—ñ—Å—Ç—å –µ–ª–µ–º–µ–Ω—Ç—ñ–≤.
